# Customer Purchase Behavior Analysis

This project analyzes customer purchasing patterns using the **Market Basket Analysis (MBA)** technique on a grocery dataset. The goal is to identify frequently bought item combinations and uncover hidden relationships between products to help businesses improve product placement, cross-selling strategies, and marketing decisions.

## Key Objectives

* Analyze customer transaction data from a grocery store dataset
* Identify frequently purchased itemsets
* Generate association rules using **Apriori Algorithm**
* Understand customer buying patterns and product relationships

## Technologies Used

* Python
* Pandas
* NumPy
* Mlxtend
* Jupyter Notebook

## Key Concepts

* Market Basket Analysis
* Association Rule Mining
* Support, Confidence, and Lift
* Data Preprocessing and Transformation

## Insights

The analysis helps retailers understand which products are often purchased together, enabling them to optimize **store layout, promotional strategies, and recommendation systems**.
